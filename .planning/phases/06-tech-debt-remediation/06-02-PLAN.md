<objective>
Replace deprecated FastAPI `@app.on_event("startup")` and `@app.on_event("shutdown")` with the modern `lifespan` async context manager pattern.

Purpose: FastAPI deprecated `on_event` in favor of the lifespan context manager (introduced in Starlette 0.20+). The current code triggers deprecation warnings and will eventually break in a future FastAPI release. The lifespan pattern is cleaner — startup runs before `yield`, shutdown runs after.

Output: `app.py` uses `@asynccontextmanager async def lifespan(app)` with `yield`, passed to `FastAPI(lifespan=lifespan)`. All existing behavior preserved.
</objective>
